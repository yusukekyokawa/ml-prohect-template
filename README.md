機械学習の実験やプロジェクトの管理方法に関するメモやノウハウをまとめる．
SIGNATEの練習問題を解きながら実際の開発環境を構築

## tips
- データはgitで管理しない．容量制限で大抵upできないから
- Dataフォルダは生データ，処理済み，データセット(訓練検証)用で構成
- パラメータ管理はhydraを使用
- 実験結果の管理はmflow or sacredを使用
- 環境はdockerで管理
